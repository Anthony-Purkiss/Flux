# Flux – Domain Specific Language Interpreter

Flux is a domain-specific language and interpreter written in Java for modelling river systems using a text-based input format.

The project was inspired by the Lox interpreter described by Robert Nystrom in *Crafting Interpreters*.

## Features

- Custom DSL syntax for defining river systems
- Interpreter written in Java
- Parsing and modelling of river networks

## Technologies

- Java

## Running the Project

Instructions for compiling and running Flux are provided in the `Compile.txt` and `Run.txt` files.

Example Flux programs can be found in the `demoPrograms` folder.  
Additional documentation, including language design tips and grammar rules, is provided in `DesignTips.txt` and `Grammar.txt`. 

## Background

This project was developed as part of the Programming Languages unit at Macquarie University.

The assignment required designing and implementing a small domain-specific language along with an interpreter written in Java.
